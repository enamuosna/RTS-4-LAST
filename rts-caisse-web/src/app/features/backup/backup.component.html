Exporter ou restaurer l'intégralité de la base PostgreSQL. Réservé aux administrateurs.

Exporter la base

Télécharge un fichier .sql contenant toutes les tables, séquences et données (caisses, opérations, journaux, utilisateurs, paramètres du reçu…). Conservez ce fichier en lieu sûr — il permet de tout restaurer en cas de problème.
